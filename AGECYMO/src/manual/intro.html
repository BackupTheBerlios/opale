Introduction

AGECYMO (Another Generalized Cylinder Modeler) est un modeleur de cylindre généralisé. Cette petite aide en ligne décrit pour chaque composant les différents raccourcis claviers accessibles et leur significations ainsi que les différentes actions possibles avec la souris.

AGECYMO supporte aussi un mécanisme de plugin d'entrée-sortie. Il peut donc être facilement étendu pour visualiser des modèles dans différents formats ou pour exporter le cylindre généralisé dans un nouveau format.

Gestion des Plugins

AGECYMO possède une gestion de plugins afin d'ajouter des fonctionnalités au logiciel. Ces plugins sont compilés séparemment. Il existe deux type de plugins : les plugins LOAD_AND_SAVE permettant de gérer les entrées sorties du logiciel (tel que l'import export en VRML) et les plugins ACTION (permettant de gerer un nouvel outil par exemple). Ces plugins sont chargés dynamiquement (seulement lorsque l'utilisateur les appelle). Ils ajoutent les boutons, icones et autres menus nécessaires à leur utilisation lors d'une opération de chargement et déchargement lors du démarrage de l'application.

Format de sauvegarde du cylindre généralisé

Le cylindre généralisé peut être sauvegardé selon un format que nous avons défini. Ainsi un fichier ".cur" est créé et contient les courbes définies par l'utilisateur dans les fenêtres chemin, section et profil. Cette sauvegarde permet de recharger ces courbes dans le futur puis de regénérer l'objet. Le cylindre généralisé créé peut aussi etre exporté au format VRML afin d'etre utilisé par d'autres modelers.
