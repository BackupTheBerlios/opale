Vue générale d'AGECYMO

Lorsque le modeleur démarre 5 fenêtres apparaissent :

  • Une fenetre sur la gauche dite principale qui contient la barre de menu et la barre d'outils ainsi qu un panneau de contrôle permettant de passer des options pour la génération du cylindre.
  • Une fenêtre 2D qui permet de définir le chemin
  • Une fenêtre 2D qui permet de définir la section
  • Une fenêtre 2D qui permet de définir le profil
  • Une fenêtre 3D qui permet soit de visualiser le cylindre généralisé produit à l'aide des courbes définies dans les fenêtres 2D soit de visualiser un objet chargé depuis un fichier grâce à un plugin.

Opérations accessibles depuis la fenêtre principale

la fenetre principale contient plusieurs menus. Le menu File permet de gérer les entrées sorties du programme et de quitter le programme.

Le menu Cylinder permet de gérer le cylindre (de le générer par exemple).

Enfin le menu help donne accès à l'aide que vous lisez actuellement.

la fenetre principale permet aussi de regler les parametres de discrétisation. l'utilisateur peut régler la résolution de la discrétisation pour le chemin (Way) et la section (Section).

Il est aussi possible de définir une torsion (via la case a cocher Torsion) et de définir un facteur d'échelle pour la section (Section Scale) et le chemin (Way Scale).

Certaines options sont accessibles via des boutons directement placés dans la fenêtre (à gauche). Ces boutons peuvent être placés par des plugins.
